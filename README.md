# 🌸 Fem-Track

**A beautiful, privacy-first menstrual cycle tracker built with modern web technologies.**

Track periods, predict future cycles, monitor symptoms, and gain insights — all while keeping your data stored locally on your device.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/fem-track.git
cd fem-track
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## 🔒 Privacy

Fem-Track is designed with privacy in mind.

- No user accounts
- No cloud storage
- No analytics
- No tracking
- No personal data collection

All information remains on your device.
